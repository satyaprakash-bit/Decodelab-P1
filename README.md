# DecodeLabs Studio – Responsive Landing Page

A modern, responsive landing page built using **HTML5, CSS3, and Vanilla JavaScript**. This project demonstrates semantic web development, mobile-first responsive design, accessibility, and interactive UI components without using any frontend frameworks.

## 🚀 Features

- Responsive mobile-first layout
- Semantic HTML5 structure
- CSS Grid & Flexbox layouts
- Modern hero section
- Responsive navigation menu
- Interactive project cards
- JavaScript counter
- Contact form with validation
- Accessible components (ARIA labels, focus states)
- Smooth scrolling navigation
- Clean and organized code

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
